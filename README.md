# ⚽ World Soccer Champions — Fan-made HTML5

Jogo de futebol de chutes ao gol inspirado no **World Soccer Champions** (Finz Games / Poki), recriado em HTML5 puro (sem dependências externas).

## 🎮 Como Jogar

| Ação | Desktop | Mobile |
|------|---------|--------|
| Mirar | Clique e arraste | Toque e arraste |
| Potência | Barra oscila enquanto arrasta | Mesma |
| Chutar | Solte o mouse | Levante o dedo |

- Mire nos **cantos superiores** para marcar **90 pontos** 🔥
- Zona central superior vale **60 pontos**
- Partes baixas valem **30 pontos**
- O goleiro se move e tenta defender!

## ⭐ Sistema de Estrelas

Cada fase tem 3 marcos de pontuação. Alcance-os para ganhar estrelas:

## 🚀 Como Rodar

Basta abrir o `index.html` em qualquer navegador moderno. Ou hospede no GitHub Pages:

1. Faça push do repositório
2. Vá em **Settings → Pages**
3. Selecione branch `main` e pasta `/` (root)
4. Acesse `https://seu-usuario.github.io/nome-do-repo`

## 🛠 Tecnologias

- **HTML5 Canvas** — renderização do campo, gol, jogadores e bola
- **Vanilla JavaScript** — toda a lógica do jogo (sem frameworks)
- **CSS3** — UI, animações e overlays
- Design responsivo para **mobile e desktop**
